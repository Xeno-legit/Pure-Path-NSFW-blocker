# Quick Start Guide - Pure Path

## 5-Minute Setup

### Step 1: Generate Icons (2 minutes)
1. Open `create-icons.html` in your browser
2. Icons will generate automatically
3. Click "Download" button for each size (16, 48, 128)
4. Save files as `icon16.png`, `icon48.png`, `icon128.png`
5. Move all three files to the `icons/` folder

### Step 2: Install Extension (2 minutes)

**Chrome/Edge:**
1. Open `chrome://extensions/`
2. Toggle "Developer mode" ON (top right)
3. Click "Load unpacked"
4. Select this folder
5. Done!

**Firefox:**
1. Open `about:debugging#/runtime/this-firefox`
2. Click "Load Temporary Add-on"
3. Select `manifest.json`
4. Done!

### Step 3: Complete Setup (1 minute)
1. Setup wizard opens automatically
2. Create a password (min 6 characters)
3. Add your goal (optional)
4. Check consent box
5. Click "Complete Setup"

### Step 4: Test It
1. Try visiting `pornhub.com` or any blocked site
2. You should see the Pure Path blocked page
3. Click the extension icon to see your stats

## Quick Tips

### View Your Stats
Click the extension icon anytime to see:
- Total sites blocked
- Days protected

### Change Password
1. Click extension icon
2. Click "Change Password"
3. Enter current and new password

### Customize Blocklists
Edit these files:
- `blocklists/domains.json` - Add domains
- `blocklists/keywords.json` - Add keywords

After editing, reload the extension.
