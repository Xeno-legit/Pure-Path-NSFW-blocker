# Icons Folder

This folder should contain the extension icons in PNG format:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create icons using:
1. Any image editor (Photoshop, GIMP, etc.)
2. Online icon generators
3. Icon design tools like Figma or Sketch

## Quick Icon Generation

You can use these free online tools:
- https://favicon.io/ (convert text/emoji to icons)
- https://www.canva.com/ (design custom icons)
- https://www.figma.com/ (professional icon design)

Simply create your design and export in three sizes: 16x16, 48x48, and 128x128 pixels.
